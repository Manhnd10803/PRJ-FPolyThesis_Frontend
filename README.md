## Clone

```zsh
 git clone https://github.com/mhfed/PRJ-FPolyThesis_Frontend.git
```

## Install dependencies

```zsh
bun install
# or
npm install
# or
yarn
# or
pnpm install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
